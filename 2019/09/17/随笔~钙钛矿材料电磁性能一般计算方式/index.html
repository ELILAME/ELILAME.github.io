钙钛矿材料电磁性能一般计算方式

我们计算钙钛矿电磁性能的时候一般通过VASP软件包完成，由于作者本人才疏学浅，故本文着重探讨计算的操作方面，对相关原理和基本理论不做详细解释。

计算方式分类： GGA 和 GGA+U

体系结构分类：Fim（亚铁磁） Fm（铁磁） AFM-（反铁磁）包括：A-（A型反铁磁） B-（B型反铁磁） C-（C型反铁磁） G-（G型反铁磁）

VASP计算基本输入文件：INCAR KPOINTS POTCAR POSCAR

计算基本流程：结构优化（晶格常数A）– 原子弛豫（各原子坐标）– 静态计算scf – 态密度dos
分别简写为： opt-isif3 – opt-isif2 – scf – dos

钙钛矿材料电磁性能计算操作流程

建立某种材料的文件夹–建立磁构型的文件夹–建立计算方式的文件夹–建立各计算流程文件夹

基本输入文件参数

例：
INCAR：
system =LaMn3Rh4O12
ISTART= 0
ICHARG= 2
PREC = high #cutoff + wrap around errors.
IBRION= 2 #conj. grad. relaxation.
NSW=551 #number of ionic steps taken in minimization. Make it odd.
ENCUT=450.000 #cutoff
EDIFF=1.0e-04
EDIFFG=-0.05
ISIF=3
ISMEAR= 0 #BZ integration method (for relaxation runs).
SIGMA= 0.2
LWAVE = .FALSE.
LCHARG = .FALSE.
LBERRY = .FALSE.
#ISYM= 0
#IALGO= 48
POTIM= 0.2
#ALGO = Normal
#LREAL= FALSE
LREAL= Auto
#NELM= 850
MAGMOM=2*0 6*6 8*6 24*0
#VOSKOWN= 1
#LORBIT = 11 #For LDOS calculation
#ICHARG = 11
ISPIN= 2
#LDAU= .TRUE.
#LDAUL= -1 2 2 -1
#LDAUU= 0 4.7 3.7 0
#LDAUJ= 0 0.7 0.7 0
#LDAUPRINT= 2
#LMAXMIX= 4
#LPARD= .TRUE.
#NBMOD= -3
#EINT= -0.5
NCORE=4

KPOINTS:
LaMn3Rh4O12
0
Monkhorest-Pack
7 7 7
0 0 0

POSCAR:
原子坐标文件，使用MS生成

POTCAR：
赝势文件：cat La/POTACR Mn/POTCAR Rh/POTCAR O/POTCAR > POTCAR

基本操作流程

GGA

  1. 将以上4个基本输入文件放入opt-isif3文件夹中，再将vasp.pbs文件导入，使用qsub vasp.pbs 提交任务。

  2. 将opt-isif3文件夹中INCAR, KPOINTS, POTCAR和CONTCAR文件导入opt-isif2文件夹中，将CONTCAR改名为POSCAR（mv CONTCAR POSCAR），修改参数
    sed -i ‘s/ISIF=3/ISIF=2/g’ INCAR
    sed -i ‘s/EDIFF=1.0e-04/EDIFF=1.0e-05/g’ INCAR
    sed -i ‘s/EDIFFG=-0.05/EDIFFG=-0.01/g’ INCAR，
    将vasp.pbs文件导入，使用qsub vasp.pbs 提交任务。

  3. 将opt-isif2文件夹中所有文件拷入scf文件夹中，修改参数
    sed -i ‘s/ISIF=2/#ISIF=2/g’ INCAR
    sed -i ‘s/EDIFFG=-0.01/#EDIFFG=-0.01/g’ INCAR
    sed -i ‘s/#LORBIT = 11/LORBIT = 11/g’ INCAR
    sed -i ‘s/POTIM= 0.2/#POTIM= 0.2/g’ INCAR
    sed -i ‘s/NSW=551/NSW=0/g’ INCAR
    sed -i ‘s/IBRION= 2/#IBRION= 2/g’ INCAR
    sed -i ‘s/LWAVE = .FALSE./LWAVE = .TURE./g’ INCAR
    sed -i ‘s/LCHARG = .FALSE./LCHARG = .TURE./g’ INCAR
    sed -i ‘s/ISMEAR= 0/ISMEAR= -5/g’ INCAR
    sed -i ‘s/SIGMA= 0.2/#SIGMA= 0.2/g’ INCAR
    将vasp.pbs文件导入，使用qsub vasp.pbs 提交任务。

  4. 将scf文件夹中所有文件拷入dos文件夹中，修改参数
    sed -i ‘s/7 7 7/10 10 10/g’ KPOINTS
    sed -i ‘s/ISTART= 0/ISTART= 1/g’ INCAR
    sed -i ‘s/ICHARG= 2/ICHARG= 11/g’ INCAR
    将vasp.pbs文件导入，使用qsub vasp.pbs 提交任务。

  5. 将scf文件夹中所有文件拷入bands文件夹中，修改参数
    sed -i ‘s/ISMEAR= -5/ISMEAR= 0/g’ INCAR
    sed -i ‘s/#SIGMA= 0.2/SIGMA= 0.2/g’ INCAR
    sed -i ‘s/ISTART= 0/ISTART= 1/g’ INCAR
    sed -i ‘s/ICHARG= 2/ICHARG= 11/g’ INCAR

    还需要修改KPOINTS文件，原本的KPOINTS如下：
    LaMn3Rh4O12 名字（随便写，不影响计算）
    0 自动产生K点
    Monkhorest-Pack 以M-P方式来自动生成K点
    7 7 7 确定沿倒格子各方向上网格点的数目
    0 0 0 对所按网格进行分割产生的K点进行平移的量

    将其修改为：
    LaMn3Rh4O12 名字（随便写，不影响计算）
    10 产生10个K点
    Line-mode 使用直线取K点模式
    rec
    10 10 10
    .0 0.5 0.0 !F 取K点的路径（可用MS构建）
    .0 0.5 0.0 !F ！这里最好三维方向都取到
    0.0 0.5 0.5 !Q ！首尾相连，形成回路
    0.0 0.5 0.5 !Q
    0.5 0.5 0.5 !M
    0.5 0.5 0.5 !M
    0.0 0.0 0.0 !G

    这里的路径可以自己修改到认为合适

GGA+U

操作方式同上，只需打开加U开关（INCAR中参数）
#LDAU= .TRUE. #控制是否加U
#LDAUL= -1 2 2 -1 #-1：s,p 轨道 #2：d,f 轨道
#LDAUU= 0 4.7 3.7 0 #LDAUU和LDAUJ为一对，差值为U值
#LDAUJ= 0 0.7 0.7 0
#LDAUPRINT= 2 #固定
#LMAXMIX= 4 #固定

将以上注释掉的参数 取消注释（删除#）–即在计算中加入了U值

本人所研究体系LaMn3Rh4O12详细计算过程

根据论文LaMn3Rh4O12: An Antiferromagnetic Quadruple Perovskite Synthesized at High Pressure，

LaMn3Rh4O12为四层钙钛矿结构，而我想要研究其磁性质，在整个体系中只有Mn和Rh对体系总体磁矩（磁性质）有影响，故需要考虑Mn和Rh的原子磁矩（Mn和Rh原子核外电子之间会有相互影响）。此处，我在画出晶体模型后我再详细解释！！

进行计算的第一步就是画出晶体结构3D图

我们所使用的软件为MS（Diamond也可以）。

  1. 打开软件，新建项目，最好保存为你所需计算的体系的名称，以后所有计算所需的不同体系结构（铁磁，反铁磁等）都保存在这个项目里。

  2. 然后新建文档和3D原子结构

  3. 开始建立晶格体系（230种空间群意味着原子在三维空间中有230种不同的排列，通过原子在14种布拉维点阵中的对称排布构成了230种空间群，主要由晶体的对称性决定！）重点！！！ 计算的基础是晶体的对称性和周期性

  4. 建立空间群

晶体的空间群和晶格常数信息均会在论文里给出（XRD扫描得到或中子衍射），一般来讲，不是做预测方面的，晶体结构都存在，若论文中未给出，可自行谷歌或参考晶体结构公开数据库http://www.crystallography.net/cod/search.html。

第二步，将之前所创建的cif文件导入VESTA打开

  1. 拖拽cif文件到VESTA

  3. 命名为POSCAR，格式为vasp（POSCAR是保存原子位置和信息的文件，之后的vasp计算中会用到）

  4. VAESTA完成

第三步，连接服务器，开始进行计算

  1. 使用Xshell新建会话

  2. 填入服务器IP地址以及登陆所需用户名和密码

  3. 连接上服务器之后 在/home/hpli目录下新建一个属于你自己的目录，如：/home/hpli/gezhizhong

  4. 接着将你所计算的所有体系都放在这个文件夹下，如：/home/hpli/gezhizhong/LaMn3Rh4O12，Pb2FeOsO6等等

  5. 最后根据你所需计算的体系不同的磁性结构进一步划分并建立文件夹，如：/home/hpli/gezhizhong/LaMn3Rh4O12/Fm，Fim，A-，B-，G-等等

第四步，vasp计算基本流程

  1. vasp所需4个基本输入文件（缺一不可）：INCAR，KPOINTS，POSCAR，POTCAR

  2. vasp一般有两种计算模式，GGA和GGA+U

  3. 在不同的磁性结构文件夹下进一步根据计算方式划分并建立文件夹，如：/home/hpli/gezhizhong/LaMn3Rh4O12/Fm/GGA，GGA+U

  4. 请将INCAR，KPOINTS，POSCAR，POTCAR 拷入GGA或者GGA+U文件夹内，然后运行一键运行脚本auto-gga或者auto-gga+u，完成后应该如：/home/hpli/gezhizhong/LaMn3Rh4O12/Fm/GGA/opt-isif3，opt-isif2，scf，dos

INCAR，KPOINTS，POSCAR，POTCAR详解

  1. POTACR

赝势文件，在~/POT/PAW_PBE/ 内可以找到所有原子所需赝势（！注意，赝势文件集成的顺序需要与结构文件，即原子的顺序一一对应），使用以下命令进行集成：

cat file1 file2 file3… > POTCAR

例：cat La/POTACR Mn/POTCAR Rh/POTCAT O/POTCAR > POTCAR

  1. POSCAR

之前所建立的晶胞结构所导出的

  1. KPOINTS

LaMn3Rh4O12 名字（随便写，不影响计算）
0 自动产生K点
Monkhorest-Pack 以M-P方式来自动生成K点
7 7 7 确定沿倒格子各方向上网格点的数目
0 0 0 对所按网格进行分割产生的K点进行平移的量

  1. INCAR

system =LaMn3Rh4O12 #名字（随便写，不影响计算）
ISTART= 0 #0：全新一次计算 #1: 读WAVECAR
ICHARG= 2 #2:从原子中读charge（初猜） #1: 读CHGCAR
PREC = high #计算精度
IBRION= 2
NSW=551 #步数
ENCUT=450.000 #截断势能
EDIFF=1.0e-04 #电子迭代收敛标准
EDIFFG=-0.05 #原子弛豫收敛标准
ISIF=3 #3：计算应力且对结构进行优化 #2：原子弛豫
ISMEAR= 0 #ISMEAR和SIGMA为一对，进行结构优化和原子弛豫时，取0和0.2 ，scf 和dos 计算时使用-5，注释掉SIGMA
SIGMA= 0.2
LWAVE = .FALSE. #计算平面波，计算scf时打开，与LCHARG为一对
LCHARG = .FALSE.
LBERRY = .FALSE.
#ISYM= 0
#IALGO= 48
POTIM= 0.2 #原子每步移动大小
#ALGO = Normal
#LREAL= FALSE
LREAL= Auto #确定投影算法是否在实空间或倒空间计算
#NELM= 850
MAGMOM=2*0 6*6 8*6 24*0 #原子数*原子理论磁矩（根据d,f轨道得到）
#VOSKOWN= 1
#LORBIT = 11 #For LDOS calculation
#ICHARG = 11
ISPIN= 2 #自旋极化(一般都须打开)
#LDAU= .TRUE. #控制是否加U
#LDAUL= -1 2 2 -1 #-1：s,p 轨道 #2：d,f 轨道
#LDAUU= 0 4.7 3.7 0 #LDAUU和LDAUJ为一对，差值为U值
#LDAUJ= 0 0.7 0.7 0
#LDAUPRINT= 2 #固定
#LMAXMIX= 4 #固定
#LPARD= .TRUE.
#NBMOD= -3
#EINT= -0.5
NCORE=4

基本完成vasp计算，使用P4vasp处理数据，用origin绘图

  1. 将VASP计算完成后的输出文件vasprun.xml导入本地

  2. 新建文件夹，将P4VASP(p4v)和vasprun.xml放入同一个文件夹中，双击打开p4v，会自动读取vasprun.xml

  3. 按个人需求打开 态密度 或者 能带图 然后导出数据
