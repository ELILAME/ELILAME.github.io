微信-青年大学习移动端破解教程

关于最近被青年大学习所困惑，光看5min的视频就算了，中间还有问题，不点还不能继续。被这种令人智熄的行为所困惑的我决定去试一试青年大学习开发人员的智商。

果然不出我所料，网页可谓相当的简洁，我不由的为开发者的智商感到深深的捉急，这智商也要让我学习？！


仔细阅读一下开发者写的代码，发现整个学习流程中间有4道插入题（应该吧，没仔细看）和8道课后习题。嗯，是用了video.pauss的语法，一点不出我的意料。：）

最后还看到了评分，嗯，这可能是以前结束学习后需要答对一定数量习题留下的。
总而言之，是网页写的不咋地，事还挺多的典型代表。
那关于破解思路就比较明了，以下是关于Android系统的破解教程，IOS我放在后面。

Android:(三步走)

1.扫码

2.进入调试页面，点击如下图所示的顶部的”信息”，勾选 打开vconsole调试功能

  1. 复制下列代码:
    Bvideo.currentTime=300；//直接跳转至视频末尾。
    Score=100;allScore=100；//以防万一
    __vconsole.style.display="none";//隐藏vconsole按钮
    然后正常打开青年大学习，点击开始学习
    再点击右下角的小绿色按钮，将复制的代码粘贴进入，点击ok即可！

IOS（不推荐h5零基础使用）：

1.IOS平台相对麻烦一些，主要是国内-微信等公司对其支持的力度有限，没有太好的前端调试工具，那在这里我推荐及我本人使用的是spy-debugger。
2.进入https://github.com/wuchangming/spy-debugger，按照作者的说明安装好调试工具，在windows环境下使用 spy-debugger -i true -w true 开始进行调试。

  1. 开始观看青年大学习，直接在调试窗口中键入
    Bvideo.currentTime=300； //直接跳转至视频末尾。
    Score=100；// 给朕把得分满山！
    回车，完成青年大学习！
